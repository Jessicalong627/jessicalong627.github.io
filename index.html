Game Show App

For this project I needed to create a word guessing game similar to Wheel of Fortune. The players had to guess a letter and if it matched then the game board would be updated. If the letter guessed was incorrect then a chance would be taken away. If the player guessed the entire word before their chances ran out a winning screen appeared and if not then a losing screen appeared. Javascript was used to determine if a letter clicked matched the game board and to decrease the counter if they guessed incorrectly. 

Skills

Javascript, Jquery, HTML, CSS

Flickr Gallery

In this project I was challenged to build an image gallery using React and the Flickr API. I displayed three links that returned a list of matching photos from Flickr. I also included a search bar that, when clicked, could be used to search for photos. I created an app using Create React App then used React Router to setup routes for the navigation. I used Axios to fetch data from the Flickr Api.
